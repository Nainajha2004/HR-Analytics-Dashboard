# HR-Analytics-Dashboard
This HR Analytics Dashboard is designed to provide a comprehensive analysis of employee data and HR performance metrics. This project aims to assist the Human Resource department in identifying workforce trends, monitoring employee attrition, and improving data-driven decision-making processes within an organization.
Objectives:
To analyze and visualize key HR metrics using Power BI.
To identify patterns and insights regarding employee retention and attrition.
To support HR decision-making through an interactive and user-friendly dashboard.
To transform raw Excel data into meaningful business intelligence insights.
The dashboard highlights the following HR metrics:
Total Employees,Active Employees,Exited Employees,Attrition Rate,Average Tenure,Average Salary,Average Performance Rating,Employee Count by Location,Gender Distribution,Hiring Trend Analysis,Employee ID Count
